# リアルタイム地震監視 RTEM  
リアルタイムで地震・津波・USGS地震情報が見れるブラウザ版のwebアプリです。  
Geminiがコードを考えたためバグがある可能性があります(なのでベータ版です)  
  
# 情報元：表示する情報  
P2P地震情報 JSON API：国内の地震情報・津波情報  
Wolfx 緊急地震速報 API：緊急地震速報(EEW)  
USGS (アメリカ地質調査所) Earthquake API：世界の地震情報  
EMSC (欧州地中海地震学センター) API：世界の地震情報(補完)  
気象庁 防災情報 (公式JSON)：兵庫県の天気予報・気温
  
その他使用しているライブラリ
Leaflet.js：地図を描画するためのメインエンジン  
CartoDB Dark Matter：地図の背景(ダークモード)のデザイン
